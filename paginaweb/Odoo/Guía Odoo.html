Guía de Instalación Odoo

Vamos a la página de descarga de Odoo, y pulsamos en la última version estable, en este caso la 15

En esta página vamos a la opción para nuestro sistema operativo, elegimos windows

Escogemos la opción de la última versión actualizada: odoo_15.0.latest.exe

Nos saltará el asistente de Instalación y simplemente seleccionamos Next>

Seleccionamos las dos opciones que aparecen marcadas y pulsamos Next>

Configuramos la información para la conexión de la base de datos y pulsamos Next>

Instalación completada

En la dirección: localhost:8069 nos pedirá que creemos la base de datos, introduciendo nuestros datos.

(Si pulsamos la caja de Demo data nos cargará datos de demostracion)

Una vez aquí, podremos ya instalar aplicaciones de Odoo que deseemos. Como demostración instalaremos la app de Sitio Web

Esta es la primera imagen que nos aparece de la aplicación dónde vamos a crear un sito web por pasos guiados, solo hay que seguir los pasos que te indiqie el asistente

Rellenamos según de que queramos crear el sitio web

En esta página nos permite elegir las características que queramos para nuestra web

Una vez creada la web lo único que hay que hacer es editarla a nuestro gusto. Tenemos un asistente con el cual nos dira paso a paso como editarla, solo hay que seguir el icono de pintura morada que hay en medio.
